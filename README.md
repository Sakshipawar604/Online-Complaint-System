# 🚀 Online Complaint System

A web-based application designed to manage and track user complaints efficiently. This system allows users to register complaints, track their status, and enables administrators to manage and resolve issues effectively.

---

## 📌 Features

* 🔐 User Authentication (Login/Signup)
* 📝 Complaint Registration
* 📊 Complaint Tracking System
* 🛠 Admin Dashboard
* 📁 Complaint History Management
* 🔎 Status Updates (Pending / In Progress / Resolved)

---

## 🛠 Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Java (Spring Boot)
* **Database:** MySQL
* **Tools:** VS Code, Git, GitHub

## ⚙️ Installation & Setup

1. Clone the repository:

   ```
   git clone https://github.com/Sakshipawar604/Online-Complaint-System.git
   ```

2. Navigate to the project directory:

   ```
   cd Online-Complaint-System
   ```

3. Open in VS Code or any IDE

4. Run the Spring Boot application:

   ```
   mvn spring-boot:run
   ```
